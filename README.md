# Portfolio Website

A simple portfolio website for a web developer and graphic designer.

## What is included

- `index.html` – landing page with hero, about, projects, skills, and contact.
- `styles.css` – responsive styling for desktop and mobile.
- `script.js` – navigation toggle and year update.

## How to use

1. Open `index.html` in your browser.
2. Replace placeholder text with your name, bio, project details, and contact info.
3. Update the project cards to reflect your actual work.
4. Add your own links for LinkedIn and GitHub.

## Customize

- Change the heading and descriptions in `index.html`.
- Adjust colors and spacing in `styles.css`.
- Add images or portfolio thumbnails by inserting `<img>` tags in the project cards.
- Save your profile photo in the portfolio folder as `profile.jpg` so it appears in the hero section.

## Notes

This is a static portfolio site that works locally and can be hosted on GitHub Pages, Netlify, or Vercel.
